As each of these exampels are in a single file, you can compile and link 
in one go for each of the files, for example

clang -Wall -Wextra -g circumference.c

-------

circumference.c - simpleI/O & computes circumference from radius

const.c - simple example with consts and prints

define_example.c - using #define.

printf_example.c - simple printf example

simple_io.c - simple scanf example.

sizes.c - shows lots of output about sizeof.
